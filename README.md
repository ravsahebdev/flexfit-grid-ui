# 🏋️ flexfit-grid-ui

A modern fitness campaign landing page built using HTML & CSS Grid.

This project was created as part of Assignment 4 from Cohort 3.0 by Sheryians Coding School while practicing CSS Grid layouts, typography hierarchy, positioning, and premium UI composition concepts.

---

## 🚀 Features

- Modern Fitness UI
- Advanced CSS Grid Layout
- Premium Card Composition
- App Promotion Sections
- Interactive UI Elements
- Gradient Backgrounds
- Modern Typography
- Aesthetic Layout Structuring

- Font Awesome
